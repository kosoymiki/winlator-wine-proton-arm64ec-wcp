# ARM64EC Patch Review

Этот файл перегенерируется скриптом `ci/proton10/arm64ec-commit-review.sh`.

В CI всегда публикуется как artifact с по-коммитным анализом:

- hash / subject / author / date
- diffstat и список файлов
- touched areas
- оценка риска и категории
- ожидаемый эффект для ARM64EC/WoW64/Winlator WCP
