# WCP Bundle / Набор WCP

## RU
- Релиз содержит `wine-11-arm64ec`, `proton-ge10-arm64ec`, `protonwine10-gamenative-arm64ec`.
- Эти пакеты публикуются как **content packages** для Aero.solator и загружаются через `Contents`.
- Каналы разделены: `stable` (тег `wcp-stable`) и rolling nightly по пакетам (`wine-11-arm64ec-latest`, `proton-ge10-arm64ec-latest`, `protonwine10-gamenative-arm64ec-latest`).
- В каждом `.wcp` включён forensic manifest (`share/wcp-forensics/*`).

## EN
- This release bundles `wine-11-arm64ec`, `proton-ge10-arm64ec`, and `protonwine10-gamenative-arm64ec`.
- These are **content packages** for Aero.solator and are downloaded via `Contents`.
- Channels are split into `stable` (tag `wcp-stable`) and per-package rolling nightly tags (`wine-11-arm64ec-latest`, `proton-ge10-arm64ec-latest`, `protonwine10-gamenative-arm64ec-latest`).
- Every `.wcp` includes a forensic manifest (`share/wcp-forensics/*`).
